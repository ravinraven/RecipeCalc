To compile:

Have all source in one folder.

$ javac *.java


To run:

$java RecipeCalc <filename>

<filename> is the name of the ingredient and recipe file.
The file has to be in a specific format. Follow TestRecipeList.txt file.

To test:

$java RecipeCalc -t

Need to TestRecipeList.txt file in the same folder.
